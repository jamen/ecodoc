
# ecodoc

> Manage docs across multiple packages

```js
var doc = ecodoc({ data: __dirname + '/cache' })

doc([ ...packages ], function (err, pkgs) {
  // ...
})
```

## Installation

```sh
$ npm install --save ecodoc
```
